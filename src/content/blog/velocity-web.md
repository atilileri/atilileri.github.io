# The Velocity of Modern Web Frameworks

Building for the web in 2026 is all about speed—both in development and in the final user experience. 

## Why Astro?

Astro's zero-JS-by-default approach is a game-changer for content-rich sites like this one. By only shipping Javascript when absolutely necessary, we ensure that the page is usable the moment it hits the screen.

```javascript
// Pure performance
export const prerender = true;
```
