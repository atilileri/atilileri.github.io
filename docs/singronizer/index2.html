Setup Required

To use this application, you need to configure OAuth credentials:

  1. Replace YOUR_SPOTIFY_CLIENT_ID in the JavaScript with your Spotify app client ID
  2. Replace YOUR_GOOGLE_CLIENT_ID with your Google OAuth client ID
  3. Configure redirect URIs in both platforms to match your domain

See the Spotify Web API and YouTube Data API documentation for setup instructions.
